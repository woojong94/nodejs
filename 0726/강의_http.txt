http 모듈 - http 웹서버를 구축해주는 내장 모듈
   				- 정적인 페이지 위주의 출력
				- 동적인 웹사이트를 만드는데 한계
				
				-> express 프레임워크 외부 모듈

웹 기본 포트
	- 80	- http로 기본 접속 포트
	- 443	- https 기본 접속 포트
	
웹서버 주소 -> 도메인
				-> localhost -> 현재 서버가 돌아가는 같은 PC, 서버에 접속할 때(127.0.0.1)
				-> 포트번호 표기
					도메인:포트번호
				
				예) localhost:8000 -> 8000번 포트에서 lcoalhost 접속
					(127.0.0.1)
					만약 포트번호가 없으면 기본 포트(http - 80, https - 443)
					

참고) 
	-> HTML이 출력 되기 전에 Response 헤더가 먼저 출력
	-> HTML이 출력 된 이후에 Response 헤더는 출력될 수 없다

	return을 하여 return 이후가 출력되는 것을 방지
